# Expense Tracker

A desktop Expense Tracker application built with Python, Tkinter, and SQLite.

## Features

- Add a new expense
- Delete a selected expense
- Store expenses permanently using SQLite
- Search expenses by title
- Filter expenses by category
- Automatically calculate total expenses
- Validate empty fields, invalid amounts, and invalid dates
- Simple and user-friendly Tkinter interface

## Technologies Used

- Python
- Tkinter
- SQLite
